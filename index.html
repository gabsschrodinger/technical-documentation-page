Introduction

"Java is a class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible. It is a general-purpose programming language intended to let application developers write once, run anywhere (WORA), meaning that compiled Java code can run on all platforms that support Java without the need for recompilation" (Wikipedia).

"The syntax of Java is similar to C and C++, but has fewer low-level facilities than either of them. The Java runtime provides dynamic capabilities (such as reflection and runtime code modification) that are typically not available in traditional compiled languages. As of 2019, Java was one of the most popular programming languages in use according to GitHub, particularly for client-server web applications, with a reported 9 million developers" (Wikipedia).

Hello World

Printing "Hello World" in Java is a little complex, compared with other programming languages like Python.

The first step is declaring the Java class:

public class HelloWorld {

}

The second step is creating the main method inside the class:

public static void main(String args[]) {

}

The last step is writing a print method:

System.out.println("Hello World");
Variables

"Variables are containers for storing data values" (w3schools).

Java variables/data types are:

  • byte: stores whole numbers from -128 to 127;
  • short: stores whole numbers from -32,768 to 32,767;
  • int: stores whole numbers from -2,147,483,648 to 2,147,483,647;
  • long: stores whole numbers from -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807;
  • float: stores fractional numbers. Sufficient for storing 6 to 7 decimal digits;
  • double: stores fractional numbers. Sufficient for storing 15 decimal digits;
  • boolean: stores true or false values;
  • char: stores a single character/letter or ASCII values.
Declaring variables

You can declare a variable in Java writing the data type value, followed by the name of the variable. You can also assign a values by using "=" [optional].

For example, this is how you can declare an integer variable called integerNumber, with a initial value of 2:

int integerNumber = 2;
Constants

"Constant is a value that cannot be changed after assigning it. Java does not directly support the constants. There is an alternative way to define the constants in Java by using the non-access modifiers static and final" (javaTpoint).

You can declare a contant by adding "final" at the beggining of the variable declaration. For example, this is how you can declare an integer constant called integerNumber, with a initial value of 2:

final int integerNumber = 2;
